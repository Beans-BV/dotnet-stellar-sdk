SimulateTransactionResponse.RestorePreamble

It can only present on successful simulation (i.e. no error) of InvokeHostFunction operations. If present, it indicates the simulation detected expired ledger entries which requires restoring with the submission of a RestoreFootprint operation before submitting the InvokeHostFunction operation. The MinResourceFee and SorobanTransactionData fields should be used to construct the transaction containing the RestoreFootprint operation.

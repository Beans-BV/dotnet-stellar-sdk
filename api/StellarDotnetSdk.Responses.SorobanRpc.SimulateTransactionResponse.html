RestorePreambleInfo

If present, it indicates that the simulation detected archived ledger entries which need to be restored before the submission of the InvokeHostFunction operation. The MinResourceFee and SorobanTransactionData fields should be used to submit a transaction containing a RestoreFootprint operation.

(optional) It can only be present on successful simulation (i.e. no error) of InvokeHostFunction operations.
